# Честный аудит: LeadDrive v2 vs. Конкуренты
### Внутренний документ для product owner | Апрель 2026
### НЕ маркетинговый материал. Основан на аудите кода.

---

## 1. Резюме для руководства

| Платформа | Сильнейшая сторона | Слабейшая сторона | Целевой рынок | Вердикт |
|---|---|---|---|---|
| **LeadDrive v2** | Финансовый блок (Cost Model, P&L, бюджеты) | Нет экосистемы, мобилки, SSO | IT-аутсорсинг, СНГ | Нишевый лидер в финансовом CRM |
| **Creatio** | No-code платформа, AI агенты | Высокая стоимость входа ($10K/год мин.) | Enterprise, средний бизнес | Лучший баланс мощности и гибкости |
| **Salesforce** | Экосистема, AI (Einstein/Agentforce) | Сложность, цена, нужны консультанты | Enterprise | Стандарт отрасли, если есть бюджет |
| **HubSpot** | Бесплатный tier, простота | Скачок цен на Enterprise | Стартапы -> средний бизнес | Лучший старт для новых компаний |
| **Pipedrive** | Визуальный pipeline, скорость внедрения | Нет поддержки, маркетинга, базовый AI | Малый бизнес, отделы продаж | Только для простых продаж |

**Общий вывод**: LeadDrive -- нишевый продукт с уникальным финансовым блоком, который не имеет аналогов в CRM-рынке. Однако по зрелости платформы, AI, экосистеме и enterprise-готовности он существенно уступает Creatio и Salesforce. Честная позиция -- "CRM + финансы для IT-аутсорсинга в СНГ за разумную цену".

## 2. Методология оценки

**Шкала**: 1-5 баллов
- **5** = лучший в классе, эталон рынка
- **4** = полный функционал с мелкими недостатками
- **3** = базовый функционал, работает для простых сценариев
- **2** = присутствует номинально, существенные пробелы
- **1** = отсутствует или непригоден

**Источники**:
- LeadDrive: аудит исходного кода (96 страниц, 2719 строк schema, 40+ API-модулей)
- Creatio: www.creatio.com (pricing, features, products)
- Salesforce, HubSpot, Pipedrive: официальные сайты, Gartner, Forrester, G2

**Принцип**: оцениваем реальную глубину реализации, а не наличие страницы в UI.

**Комментарий -- самый важный для честности**:

AI в LeadDrive **работает** на уровне скоринга лидов (через Claude API, с логированием стоимости, rate limiting). Есть AI Command Center с конфигурацией агентов, чат-сессиями, guardrails.

Но при аудите кода обнаружено: AI Sales Assistant, AI Support Agent, AI Command Center -- это в основном **UI-оболочки с ограниченной глубиной**. Конфигурация агентов сохраняется в БД, чат работает, но нет:
- Автономных AI-агентов (как Agentforce или Creatio AI Agents)
- Предиктивной аналитики
- AI-рекомендаций по next best action
- Автоматической обработки тикетов

Creatio имеет 15+ предварительно обученных AI-агентов. Salesforce Agentforce -- автономные агенты с доступом к данным. LeadDrive -- базовый чат с Claude API.

**Комментарий -- это ЕДИНСТВЕННАЯ категория, где LeadDrive безоговорочно лидирует**:

Ни один из конкурентов не имеет встроенного Cost Model Engine, бюджетирования с план/факт анализом, анализа прибыльности по клиентам, динамического ценообразования. В CRM-мире это обычно решается через интеграцию с ERP (SAP, 1C, NetSuite). LeadDrive закрывает это из коробки.

Для IT-аутсорсинговых компаний (целевой рынок) это реальная ценность: понимание маржинальности проектов, контроль бюджетов, прогнозирование cashflow.

Подтверждено аудитом: 19 API-роутов только для бюджетирования, отдельный модуль cost-model с AI-анализом, экспорт в Excel.

## 6. SWOT-анализ LeadDrive v2

### Strengths (Сильные стороны)
- Уникальный финансовый блок -- нет аналогов в CRM
- MTM модуль для полевых команд
- All-in-one: CRM + ERP + Finance без интеграций
- Самая низкая TCO для 25+ пользователей
- СНГ-ориентированность (AZ/RU/EN, Telegram, VK)
- Self-hosted: контроль данных и compliance
- Работающий AI-скоринг лидов (Claude API)

### Weaknesses (Слабые стороны)
- Нет мобильного приложения
- Нет экосистемы интеграций (0 marketplace)
- AI неглубокий (кроме скоринга)
- 0 тестов -- высокий риск регрессий
- Нет SSO/LDAP -- блокер для enterprise
- Нет no-code платформы
- Нет публичной API-документации
- Зависимость от малой команды разработки

### Opportunities (Возможности)
- PWA / мобильное приложение
- Выход на рынок СНГ (Казахстан, Узбекистан, Грузия)
- Углубление AI (реальные агенты, не UI-заглушки)
- White-label / OEM для партнёров
- Marketplace хотя бы для 10-20 базовых интеграций
- Вертикальные решения (консалтинг, staffing, digital agencies)

### Threats (Угрозы)
- HubSpot Free убирает стартапы из воронки
- AI гонка (Agentforce, Creatio AI) ускоряется экспоненциально
- Отсутствие тестов -> рост техдолга -> замедление разработки
- Зависимость от одного рынка (Азербайджан)
- Creatio активно работает на СНГ-рынке
- Появление вертикальных CRM для аутсорсинга от конкурентов

## 7. Позиционирование: рекомендации

### 7.1 Целевой сегмент
IT-аутсорсинговые и сервисные компании 20-100 человек в СНГ, которым нужен CRM + финансовый контроль проектной маржинальности без Enterprise-бюджета.

### 7.2 НЕ конкурировать с:
- **Salesforce** в enterprise (нет шансов без SSO, SOC2, экосистемы)
- **HubSpot** в стартапах (бесплатный tier не побить)
- **Creatio** в no-code/automation (годы отставания)
- **Pipedrive** в простоте (другая философия)

### 7.3 Приоритеты развития

| Приоритет | Задача | Почему | Усилия |
|---|---|---|---|
| **P0** | Мобильное приложение (PWA) | Без мобилки нет полевых продаж | 2-4 недели |
| **P0** | Автотесты (хотя бы critical paths) | 0 тестов = невозможен рефакторинг | 2-3 недели |
| **P1** | SSO/LDAP | Блокер для enterprise-продаж | 1-2 недели |
| **P1** | Углубить AI (реальные агенты) | Рынок движется к AI-first CRM | 4-8 недель |
| **P2** | Публичная API документация | Без неё нет интеграций | 1-2 недели |
| **P2** | 10 базовых интеграций (Zapier, 1C, Slack) | Минимальная экосистема | 4-6 недель |
| **P3** | No-code workflow builder | Конкурентная необходимость | 8-12 недель |

### 7.4 Ключевое позиционирование (честное)
